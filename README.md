
# Python To-Do List Application

A simple command-line To-Do List application built using Python.  
This project allows users to manage daily tasks efficiently with persistent storage.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using JSON file storage
- Basic error handling

## Technologies Used
- Python
- JSON
- File Handling

## Project Use Case
